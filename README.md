## Project Overview
An embedded and database-driven system designed to monitor and optimize parking lot usage across various environments such as campuses, malls, airports, or city centers. The system integrates IoT sensors, cameras, and user interfaces to detect vehicle presence, track parking availability, and manage payments in real time.
